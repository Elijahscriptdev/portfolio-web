Stay In Touch

This is a full-stack website built with HTML, CSS, Ruby, Ruby On Rails, SASS, Cloudinary

Stay In Touch is a facebook clone website where users can signup to create posts, make new friends, comment on any post of choice, like posts and upload images.

Functionalities

  • Signup
  • Login
  • Create Posts
  • upload images
  • Add friend
  • Unfriend friend
  • Confirm friends
  • Like posts
  • Upload images
  • View post of Friends only
